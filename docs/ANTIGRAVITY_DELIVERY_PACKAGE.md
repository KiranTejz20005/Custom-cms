# 🚨 CRITICAL REMINDERS FOR ANTIGRAVITY

⚠️ **Before you write any code:**

1. **NO Strapi dependency**
   - Don't create Strapi APIs
   - Don't call Strapi endpoints
   - Only use Xano

2. **grade_ids must be array**
   - [6, 7, 8] ✅
   - "6,7,8" ❌

3. **API endpoint names exact**
   - GET /get_all_courses (not /courses)
   - POST /mappings (not /create-mapping)
   - Check QUICK_REFERENCE.md for exact paths

4. **Test with Postman first**
   - Before UI, test all Xano endpoints
   - Verify responses match spec

5. **Error handling critical**
   - Try-catch all API calls
   - User-friendly error messages
   - Log errors for debugging

6. **No client-side filtering**
   - Xano filters data
   - Frontend just displays
